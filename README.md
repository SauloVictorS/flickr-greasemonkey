Flickr Greasemonkey
===================

Userscripts to enhance Flickr.

Installation
------------
1. Install [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/)
2. Visit the add-on page for whichever script you want.
3. Hit Install
